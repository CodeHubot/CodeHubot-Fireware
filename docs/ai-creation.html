🎯 教学要点

1. 提示词工程

  • 角色定义：明确智能体的身份和能力边界
  • 示例驱动：通过示例对话引导模型行为
  • 参数约束：明确参数类型和格式要求
  • 错误处理：指导模型如何处理异常情况

2. 插件系统

  • OpenAPI规范：理解RESTful API设计
  • 设备控制：学习如何通过API控制硬件
  • 数据查询：掌握传感器数据的获取与处理
  • 插件扩展：自定义插件开发与集成

3. 知识库与RAG

  • 文档处理：文本分块、向量化存储
  • 检索策略：相似度搜索、上下文注入
  • 增强生成：结合检索结果生成回答

4. 工作流编排

  • 节点设计：LLM节点、插件节点、条件节点
  • 流程控制：条件分支、循环、并行执行
  • 调试技巧：单步执行、日志查看、错误定位
